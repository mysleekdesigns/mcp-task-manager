## Responsive Behavior

### Desktop (1024px+)
- Modals: 600px width, centered
- Settings page: 896px max width (max-w-4xl)
- Full sidebar visible
- All features accessible

### Tablet (768px - 1023px)
- Modals: 90% width, centered
- Settings page: Full width with padding
- Collapsible sidebar
- All features accessible

### Mobile (< 768px)
- Modals: Full screen with padding
- Settings page: Full width, single column
- Hidden sidebar (hamburger menu)
- Forms stack vertically
- Buttons full width

## Accessibility Features

### Keyboard Navigation
- Tab: Move between fields
- Shift+Tab: Move backward
- Enter: Submit form / Confirm action
- Escape: Close modal / Cancel action
- Space: Toggle checkboxes
- Arrow keys: Navigate dropdowns

### Screen Reader Support
- All form fields have associated labels
- Error messages announced on validation
- Loading states announced
- Success/error toasts announced
- Modal focus trap (can't tab outside)
- ARIA labels on icon-only buttons

### Focus Indicators
- Blue ring on focused elements
- Visible focus on all interactive elements
- Focus restored after modal closes
